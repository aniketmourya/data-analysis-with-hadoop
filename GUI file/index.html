About the project?

The poject is processing large unstructured retail web crawling dataset which is excel format using MapReduce. After processing this data stored on HDFS in multiple file according to different condition for analysis using dashboard. Using Hive component of hadoop it fire advanced query on this dataset and using Sqoop component of Hadoop it exports this data into relational database to perform Prediction and Classification over the web crawling data. In this dashboard you will see the list of various item classified into different categories based on the input.
